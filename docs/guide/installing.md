# Начало работы

Если вы хотите поэкспериментировать только с VuePress, вы можете установить его глобально:

```sh
# install globally
yarn global add vuepress # OR npm install -g vuepress

# create a markdown file
echo '# Hello VuePress' > README.md

# start writing
vuepress dev

# build
vuepress build
```
